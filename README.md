# Aplicativo Pequenos Agricultores

## Objetivo

O **Aplicativo Pequenos Agricultores** foi desenvolvido com o objetivo de fornecer uma plataforma simples e prática para pequenos agricultores gerenciarem suas plantações. O app oferece funcionalidades para registrar diferentes tipos de cultivos, acompanhar informações sobre o clima e otimizar o processo de gestão agrícola.

Com esse aplicativo, o agricultor pode:
- Adicionar novos cultivos à sua lista.
- Visualizar detalhes sobre cada cultivo, como informações climáticas.
- Acompanhar as condições ideais para a produtividade das culturas.

O aplicativo é voltado para pequenos agricultores que buscam melhorar a eficiência e a organização na gestão de suas plantações, utilizando uma interface simples e acessível.

## Funcionalidades

- **Tela Inicial**: Exibe uma lista de cultivos já cadastrados com opções para visualizar detalhes e adicionar novos cultivos.
- **Tela de Cadastro de Cultivo**: Permite que o agricultor registre novas plantações, fornecendo o nome do cultivo.
- **Tela de Detalhes**: Exibe informações detalhadas sobre o cultivo selecionado, como clima, temperatura e previsões de chuva.
- **Interface Simples e Intuitiva**: Design focado em facilitar o uso, mesmo para agricultores sem experiência em tecnologia.

## Como Usar

1. **Iniciar o App**: Ao abrir o aplicativo, você verá a tela inicial com uma lista de seus cultivos cadastrados.
2. **Adicionar um Novo Cultivo**:
   - Toque no ícone de **"+"** no canto superior direito da tela inicial.
   - Insira o nome do cultivo e toque em **Adicionar**.
3. **Visualizar Detalhes de um Cultivo**:
   - Na tela inicial, toque no nome de qualquer cultivo para ver mais detalhes, como clima, temperatura e previsões de chuva.
4. **Gerenciamento**: Os cultivos podem ser visualizados e gerenciados, facilitando a organização do agricultor.

## Como Rodar o Projeto

### Pré-requisitos

Certifique-se de que você tenha o Flutter instalado em seu ambiente de desenvolvimento. Caso ainda não tenha, siga as instruções no [site oficial do Flutter](https://flutter.dev/docs/get-started/install).

### Rodando o App

1. Clone o repositório:
    ```bash
    git clone https://github.com/seu-usuario/app-pequenos-agricultores.git
    ```

2. Navegue até a pasta do projeto:
    ```bash
    cd app-pequenos-agricultores
    ```

3. Instale as dependências:
    ```bash
    flutter pub get
    ```

4. Execute o aplicativo no seu dispositivo ou emulador:
    ```bash
    flutter run
    ```
